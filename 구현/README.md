# 구현 & 시뮬레이션

특정 알고리즘을 알아야 풀리는 문제는 아니기 때문에 난이도가 비교적 쉬운 편이다. 그러나 복잡한 시뮬레이션 문제의 경우 원리를 제대로 이해해야 풀 수 있는 어려운 문제들도 존재한다. 시뮬레이션을 머릿속으로 그려보도록 하자. 머릿속에서 막힘없이 시뮬레이션 될 경우 대부분의 문제가 풀린다.
